### Kết quả đạt được tuần 4:
* Củng cố và chuẩn hóa quy trình sao lưu, phục hồi dữ liệu tự động cho nhiều dịch vụ khác nhau trên nền tảng AWS.
* Tự thiết lập các chính sách lưu giữ (Backup Rule) và kho chứa bản sao lưu (Backup Vault) tuân thủ chặt chẽ yêu cầu hệ thống.
* Nắm vững cơ chế xử lý lỗi và giám sát hệ thống tự động nhờ tích hợp thông báo từ Amazon SNS vào tiến trình backup.
* Hiểu rõ khái niệm Hybrid Storage, cấu hình thành công Amazon FSx for Windows File Server và mount File Share thành công trên máy tính cục bộ.
