### Week 1 Objectives:
* Understand AWS overview, create an account, and set up billing/budgets.
* Get familiar with the AWS Management Console and access management using IAM.

### Week 1 Achievements:
* Gained a solid understanding of cloud computing concepts and core AWS service groups (Compute, Storage, Networking, Database).
* Successfully created and secured an AWS Free Tier account, and set up cost alerts (AWS Budgets) to control expenses during practice.
* Mastered the AWS Management Console, experimented with CloudShell, and learned how to quickly search and manage services.
* Completed IAM setup by creating Users, Groups, and Roles, applying the principle of least privilege, and mastering Switch Role for secure access.
